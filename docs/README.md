# Документация по библиотеке geometric_lib
Данная библиотека содержит в себе методы работы для вычесления параметров простейших геометрических фигур, таких как: квадрат, треугольник, прямоугольнких и круг

## rectangle.py
Файл для работы с прямоугольником.

Имеется 2 функции:

**area(a: int, b: int): int** - для получения площади прямоугольника. Ожидает длины 2-х непротивоположных сторон прямоугольника и возвращает площадь данного прямоугольника.

**perimeter(a: int, b: int): int** - для получения периметра прямоугольника. Ожидает длины 2-х непротивоположных сторон прямоугольника и возвращает периметр данного прямоугольника

## triangle.py
Файл для работы с треугольником.

Имеется 2 функции:

**area(a: int, h: int): int** - для получения площади треугольника. Ожидает длину основание и длину высоты и возвращает площадь данного треугольника.

**perimeter(a: int, b: int, c: int): int** - для получения периметра треугольника. Ожидает длины всех сторон треугольника и возвращает периметр данного треугольника

## square.py
Файл для работы с квадратом.

Имеется 2 функции:

**area(a: int): int** - для получения площади квадрата. Ожидает длину стороны и возвращает площадь данного квадрата.

**perimeter(a: int): int** - для получения периметра квадрата. Ожидает длину стороны и возвращает периметр данного квадрата

## circle.py
Файл для работы с кругом.

Имеется 2 функции:

**area(r: int): float** - для получения площади круга. Ожидает длину радиуса и возвращает площадь данного круга.

**perimeter(r: int): float** - для получения периметра круга. Ожидает длину радиуса и возвращает периметр данного круга
