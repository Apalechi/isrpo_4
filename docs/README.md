# Документация по библиотеке geometric_lib
Данная библиотека содержит в себе методы работы для вычесления параметров простейших геометрических фигур, таких как: квадрат, треугольник, прямоугольнких и круг

## rectangle.py
Файл для работы с прямоугольником.

Имеется 2 функции:

**area(a: int, b: int): int** - Возвращает площадь прямоугольника.
```
    Параметры:
            a (число): длина
            b (число): высота

    Возвращаемое значение:
            area (число): числовое значение площади прямоугольника
```
_пример использования_: ```a = area(1, 5) #a = 5```
    

**perimeter(a: int, b: int): int** - Возвращает периметр прямоугольника.
```
    Параметры:
            a (число): длина
            b (число): высота

    Возвращаемое значение:
            perimeter (число): числовое значение площади прямоугольника
```
_пример использования_: ```a = perimeter(1, 5) #a = 12```


## triangle.py
Файл для работы с треугольником.

Имеется 2 функции:

**area(a: int, h: int): int** -  Возвращает площадь треугольника.
```
    Параметры:
            a (число): длина основания
            h (число): высота

    Возвращаемое значение:
            area (число): числовое значение площади треугольника
```
_пример использования_: ```a = area(5, 2) #a = 5```


**perimeter(a: int, b: int, c: int): int** - Возвращает периметр треугольника.
```
    Параметры:
            a (число): первая сторона
            b (число): вторая сторона
		        c (число): третья сторона

    Возвращаемое значение:
            perimeter (число): числовое значение площади треугольника
```
_пример использования_: ```a = perimeter(5, 2, 3) #a = 10```
